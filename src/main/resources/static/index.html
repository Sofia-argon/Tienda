Caracteristicas

Alambres o cables de CCA, con un aislamiento termoplástico de PVC y una cubierta termoplástica de nylon para temperatura máxima de operación de 90°C.

Utilizado en conductos o sobre bandejas para aplicaciones residenciales, comerciales e industriales, en lugares secos, THHN.

Utilizado para conexión de motores y tableros de control, para alambrado de circuitos de energía, iluminación y electrodomésticos.

Temperatura: 90° C

Voltaje: 600V

Normas: se fabrican tomando como referencia las normas ASTM B566, UL-83 y UL-1581.
